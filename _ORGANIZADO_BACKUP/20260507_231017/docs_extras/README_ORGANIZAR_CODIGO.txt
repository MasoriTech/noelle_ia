Como usar:

1. Copie ORGANIZAR_CODIGO_NOELLE.bat para a raiz do projeto Noelle.
2. Rode o .bat pela raiz do projeto.
3. Ele organiza docs, reforça .gitignore e remove pastas que não devem ir ao GitHub.
4. Depois abra o GitHub Desktop e confira os Changes antes de commitar.

Nao commite se aparecer node_modules, logs, release, build, dist, modelos ou cache.
