# Noelle V19.8.9a — Mega UI/Settings Reforço 2026

Este patch reforça o V19.8.9 antes da aplicação.

## Objetivo

- Remover duplicidade entre V19.8.9 e V19.8.9a.
- Bloquear com mais força os overlays antigos `Avatar Lab` e `Room V19`.
- Melhorar Configurações com cards úteis.
- Preservar Chat IA, Avatar, Room, Widget, VRM, VRMA, PNG e GLB.
- Manter `iniciar.bat` único, com opção `[1]` apenas iniciando o app.

## Ordem

1. Copie o pack para a raiz do projeto.
2. Rode `iniciar.bat`.
3. Use `[3] Reparar/aplicar reforco Mega UI + Configuracoes V19.8.9a`.
4. Use `[2] Rodar diagnostico V19.8.9a`.
5. Use `[1] Iniciar programa agora`.
