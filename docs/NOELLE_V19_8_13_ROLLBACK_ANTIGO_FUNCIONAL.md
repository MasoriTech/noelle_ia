# Noelle/Yoru V19.8.13 — Rollback antigo funcional

Este pack desfaz os patches agressivos de Configurações/tema que deixaram a janela principal vazia.

Ele procura automaticamente backups de `src/controls.html` e escolhe o mais provável de ser funcional, priorizando versões anteriores aos dashboards V19.8.11/11a/11b/11c/11d/12.

## Como usar

1. Copie o pack para a raiz do projeto.
2. Rode `iniciar.bat`.
3. Escolha `[3] Restaurar antigo funcional`.
4. Rode `[2] Diagnóstico`.
5. Feche e abra pela opção `[1]`.
