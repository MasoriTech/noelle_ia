# Noelle V19.2.1 — Settings/About/Room Fix

Esse patch corrige o que ficou faltando no V19.2.

## Correções

- Configurações/Sobre agora são injetadas também via preload.
- Remove botão flutuante antigo "Room".
- Remove botão flutuante antigo "Room V19".
- Cria um único botão canônico "Room V19".
- Mantém Chat IA sem mexer.
- Configurações recebe as 30 opções discutidas.
- Sobre recebe informações do projeto, links, formatos, licenças e tecnologias.

## Importante

Depois de aplicar, reinicie a Noelle. O preload precisa recarregar para injetar o runtime.

## Uso

```bat
INICIAR.bat
```

Escolha:

```txt
[1] Aplicar V19.2.1 e iniciar Noelle
```

## Teste

1. Abrir Noelle.
2. Confirmar que existe apenas um botão Room V19.
3. Abrir Configurações.
4. Ver "Configurações avançadas V19.2.1".
5. Abrir Sobre.
6. Ver links e informações extras.
