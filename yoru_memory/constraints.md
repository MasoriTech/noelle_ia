# Yoru - constraints.md

Limites e proteções.

- Não salvar senhas, tokens, chaves ou dados privados sensíveis em texto puro.
- Não sobrescrever memórias existentes sem backup.
- Não depender de internet para iniciar o modo básico.
- Não bloquear o programa se Ollama/voz/avatar estiver indisponível.
- Separar estado temporário de memória permanente.
- Logs devem ajudar diagnóstico, não expor segredo.
