# Yoru - user.md

Preferências do usuário para a Yoru.

- Preferência: packs com poucos arquivos .bat.
- Preferência: sempre incluir iniciar.bat atualizado.
- Preferência: Yoru com foco em modo widget por padrão quando possível.
- Preferência: terminal com mensagens claras de carregamento, pronto, erro e fallback.
