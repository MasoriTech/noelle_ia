# Como Testar

1. Abra `iniciar.bat`.
2. Escolha `2. Diagnóstico`.
3. Se Ollama estiver online, escolha `1. Iniciar app`.
4. No app, clique em `Diagnosticar Ollama`.
5. Envie: `oi`.

Resultado esperado:

- App não trava.
- Mostra resposta da Yoru.
- Se der erro, aparece na interface sem derrubar a janela.
